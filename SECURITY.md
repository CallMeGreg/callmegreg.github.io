# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it using [GitHub's private vulnerability reporting](https://github.com/callmegreg/callmegreg.github.io/security/advisories/new) feature. This ensures the issue is handled confidentially and responsibly.

**Please do not open a public issue for security vulnerabilities.**

## What to Include

When reporting, please provide as much of the following as possible:

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact
- Any suggested fixes (optional)

## Response

Reports will be acknowledged and responded to as soon as possible. You can expect:

1. Confirmation that the report was received
2. An assessment of the vulnerability
3. A plan for remediation, if applicable

Thank you for helping keep this project safe.
